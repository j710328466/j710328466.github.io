## 背景

随着年龄增大，工作经验的丰富，就开始想着能不能多赚点钱。

正好最近来了个外包的咨询，我就开始想着是不是可以开始着手去建立一个 SOP 来方便以后接项目，提升沟通效率

## 流程

### 第一步：需求了解

一定要明确了解一下几个点：

1. 你们是否有 ui 稿？
2. 是纯前端需求，还是要带后端？
3. 大概工作周期？
4. 大概预算？

需要思考一下怎么留住客户，并且给客户一种自己赚到的感觉，不然会流失
