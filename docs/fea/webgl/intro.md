# 💊 webGL

## 介绍

### 基本对象

- 场景：scence
- 相机：camera
- 相机创建实例图（THREE.PerspectiveCamera(fovy, aspect, zNear, zFar)）：
- 渲染器：renderer

### 绘制方法

- 网孔（Meshes，推荐）
- 多边形（Polygons）
- 顶点（Vertices）

### 其它

- 材料（Materials）
- 纹理（Textures）
